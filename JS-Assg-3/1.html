Difference

What is the difference between following 2 functions if any? 
				
function checkAge(age) {
  if (age > 18) {
    return true;
  } else {
    return confirm('Did parents allow you?');
  }
}

function checkAge(age) {
  if (age > 18) {
    return true;
  }
 return confirm('Did parents allow you?');
}
ANSWER:
-Both the function's aim is same i.e. checking if the age is less than 18 then returning true, otherwise confirm from the user 'Did parents allow you?' -The difference is, in the 1st function pure 'if-else' block has been written and in 2nd function, only 'if' block is written, so if it enters the 'if' block, it will return true and the return statement after the 'if' block will never be executed and if it doesn't go inside the 'if' block it will execute the return statement written outside the 'if' block.
